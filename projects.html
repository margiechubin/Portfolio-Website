HackIllinois Project 2015


For my team's project, Restaurandom, we wanted to ease the difficulty for chronically indecisive folks of choosing where to eat. Using an API, we give the user information such as the cuisine or price level of great restaurants near a given location. One may enter a zip code or city, and get a "restaurandomized" suggestion based on popularity.

Fundraising Account Checker

In high school, I was on the Dance Marathon exec board. Sensing a lack of communication between the Executive Board and the fundraising students, I designed an online application using Google's API that accesses the fundraising data from private google sheets. Rather than ask someone to look up information, the information is available through the application. It has been used for a few years now and has over 3000 views.

Fruit Frenzy Game


I have also written a few games, one can be seen here. I wrote this simple game as a free choice project in my video game programming class in high school. We had a few weeks to implement any game we wanted, with a mobile focus in mind. Written in C#, it was meant to be my version of another game I had found online. While it can be played by clicking the fruit quickly on a computer, the real fun comes with the touch screen mobile phone version, where the higher levels can be reached.
